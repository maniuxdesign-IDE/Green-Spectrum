    Questions about setting up your journey

    Why do you need the organisation website?

    It may help identify publicly available context later. Green Spectrum should never treat public information as verified internal evidence without confirmation.

    What if I do not know an answer?

    Select “Not sure” or skip optional questions. Uncertainty should be recorded rather than hidden.

    Do I need to upload reports?

    No. Reports can improve context, but the journey can begin without them.

    Will this replace the Explore stage?

    No. Onboarding provides a brief starting context. Explore conducts the deeper Business, Human and Planetary enquiry.

    Can I change organisation details later?

    Yes. The organisation profile remains editable throughout the journey.

    Can I create more than one journey for the same organisation?

    Yes. One organisation may contain multiple journeys focused on different challenges.

    When should I choose Facilitation Mode?

    Choose it when the challenge requires participation, specialist knowledge, shared interpretation or collective ownership.

    Profile, privacy and access for this test journey.

    This version is designed for user testing without accounts. Your journey is held as a local guest session so testers can get straight to the workflow.

    Profile

    Your working profile is created from the onboarding answers, role, organisation context and journey mode. No sign-in is required for this MVP.

    Privacy

    Only enter information you are comfortable using in a test environment. The prototype stores journey responses locally for the session and uses them to carry context between pages.

    Accessibility

    The MVP supports keyboard navigation, visible focus states, semantic sections and reduced-motion preferences. User-test feedback should flag any barriers.
